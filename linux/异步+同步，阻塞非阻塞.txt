1:什么是异步非阻塞，同步和异步

同步和异步侧重通信机制比如nginx 可以nginx可以接受n个客户端同时链接，实现方式：多进场 or 多线程 or epoll
阻塞非阻塞分为阻塞调用和阻塞io：epoll——wait 可以阻塞调用也可非阻塞调用，阻塞io和非阻塞io通过no_nlock 设置

2：多路复用：提高io的使用效率，多路复用可以同时处理阻塞io和非阻塞io


3:阻塞io多路复用例子
$clients = epoll_wait()
foreach($client as $c) {
    $str = fread($c);
}
非阻塞io多路复用例子
$clients = epoll_wait()
foreach($client as $c) {
    while(1) {
       $str = fread($c);
    }
}
4: 关于epoll 两种模式说明

epoll有两种工作模式：LT（水平触发）模式和ET（边缘触发）模式。

默认情况下，epoll采用 LT模式工作，这时可以处理阻塞和非阻塞套接字，而上表中的 EPOLLET表示可以将一个事件改为 ET模式。ET模式的效率要比 LT模式高，它只支持非阻塞套接字。


5:nginx为什么是异步非阻塞的

nginx非阻塞：多路复用器可以解决阻塞问题
异步怎么理解：可以接受多个client 链接
